# Given-When-Then 速查

| 关键字 | 用途 | 示例 |
|--------|------|------|
| Given | 前置条件 | Given 用户已登录 |
| When | 触发操作 | When 用户点击提交 |
| Then | 预期结果 | Then 页面显示"保存成功" |
| And | 补充条件 | And 购物车中有 3 件商品 |
| But | 排除条件 | But 不发送通知邮件 |
